# Google Form Clone (HTML, CSS & JavaScript)

A fully responsive **Google Form clone** built using **HTML, CSS, and Vanilla JavaScript**, featuring complete **client-side validations** and a clean user interface.

📍 **Live Demo:** https://google-form-nine.vercel.app/

## ✨ Features

- ✅ Google Form–like UI
- ✅ Fully responsive design
- ✅ Client-side form validation
- ✅ Required field validation
- ✅ Email format validation
- ✅ Phone number validation
- ✅ Error messages for invalid inputs
- ✅ Clean and user-friendly layout

---

## 🛠️ Tech Stack

- **HTML5** – Structure  
- **CSS3** – Styling & responsiveness  
- **JavaScript (ES6)** – Form validation & logic
