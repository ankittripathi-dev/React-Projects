# 🏥 Healthcare Admin Dashboard

A B2B Healthcare Admin Dashboard built to manage and visualize healthcare-related data.
The application focuses on clean UI, proper state management, authentication,
and a scalable frontend architecture using modern React tooling.

## ✨ Features

- Authentication with email and password (dummy credentials)
- Protected routes for authenticated users
- Dashboard overview with key healthcare metrics
- Global state management using Redux Toolkit
- Reusable and accessible UI components using Radix UI
- Responsive layout for different screen sizes
- Logout functionality

## 🚀 Getting Started

### Prerequisites
- Node.js (v18 or above recommended)
- npm

### Installation & Setup
```
Clone the repository  
git clone https://github.com/Adityaguptawebdev/healthcare-admin-dashboard.git

Navigate to the project directory  
cd healthcare-admin-dashboard

Install dependencies  

npm install

Start the development server  
npm run dev
```
