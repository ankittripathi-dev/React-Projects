## 🧩 About the Project

Many beginners struggle to design a professional portfolio from scratch.  
This platform solves that problem by offering **multiple modern portfolio templates** that are easy to preview, understand, and customize.

Each template is built using **pure HTML, CSS, and JavaScript**, making it beginner-friendly and easy to deploy.

---

## 🧠 How It Works

1. Users browse from **20+ portfolio templates**
2. Each template has a **live preview**
3. Users can view and reuse **HTML, CSS, and JS code**
4. Templates are fully **responsive and customizable**
5. No frameworks required — simple and fast
