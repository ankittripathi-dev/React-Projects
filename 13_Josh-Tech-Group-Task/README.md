# 🍕 Frontend Mockup Project  

A responsive **frontend project** built using **HTML, CSS, and JavaScript**. The design was implemented from a **Figma mockup** with a pixel-perfect approach.  

- Live link: https://jtgeats-beryl.vercel.app/   
---

## 🚀 Features  
- Clean and semantic **HTML5** structure  
- Responsive **CSS3** styling  
- Interactive elements with **Vanilla JavaScript**  
- Includes assets and icons  
- Cross-browser compatibility  

---

## 📂 Project Structure  
- `index.html`  
- `style.css`  
- `script.js`  
- `images/`
